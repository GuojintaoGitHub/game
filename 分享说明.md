# 俄罗斯方块 - 分享给他人使用

游戏是**单个 HTML 文件**（`tetris.html`），无需安装、无需服务器，用浏览器即可运行。

---

## 方式一：直接发文件（最简单）

1. **把 `tetris.html` 发给对方**（微信、邮件、U 盘等）。
2. 对方用**电脑或手机浏览器**打开这个文件即可玩。
   - 电脑：双击文件，或用浏览器菜单「文件 → 打开」选择该文件。
   - 手机：把文件传到手机后，用浏览器打开，或从文件管理器里点开（选择用浏览器打开）。

**优点**：不依赖网络，收到文件就能玩。  
**注意**：首次打开时若联网，会加载一个字体，界面更美观；断网时用系统默认字体，不影响游戏。

---

## 方式二：发链接（在线玩）

希望对方点一个链接就能玩，可以把游戏放到免费网页托管上。

### 使用 GitHub Pages（适合有 GitHub 账号）

1. 注册 [GitHub](https://github.com)（若没有）。
2. 新建一个仓库（Repository），名字随意，例如 `tetris-game`。
3. 把本地的 `tetris.html` 上传到仓库里（可改名为 `index.html`，这样访问地址更短）。
4. 在仓库里点 **Settings → Pages**：
   - Source 选 **Deploy from a branch**
   - Branch 选 **main**，文件夹选 **/ (root)**，保存。
5. 等一两分钟后，在 Pages 页面会看到访问地址，例如：  
   `https://你的用户名.github.io/tetris-game/`  
   把这个链接发给别人即可。

### 使用 Netlify Drop（不用写代码）

1. 打开 [https://app.netlify.com/drop](https://app.netlify.com/drop)。
2. 把 **整个项目文件夹**（里面有 `tetris.html` 的那个文件夹）拖进网页。
3. Netlify 会生成一个随机链接，例如 `https://xxx.netlify.app`，分享这个链接即可。

---

## 小结

| 方式       | 适用场景           | 对方需要做什么     |
|------------|--------------------|--------------------|
| 发 HTML 文件 | 熟人、内网、离线   | 用浏览器打开文件   |
| GitHub Pages | 想有一个固定链接   | 点链接即可         |
| Netlify Drop | 想快速上线、不发文件 | 点链接即可         |

游戏不收集任何数据，所有逻辑都在本地浏览器中运行，对方只要能用浏览器打开 HTML 或访问你给的链接，就可以玩。
